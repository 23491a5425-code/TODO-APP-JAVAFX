# 📝 TODO APP

## ✨ FEATURES

📌 Task Management 
1. Create new tasks with a title and description.
2. Update existing tasks.
3. Delete tasks that are no longer needed.
4. View all tasks in a structured list.

🚦 Task Status Tracking 
1. To Do – Task has not been started.
2. In Progress – Task is currently being worked on.
3. Done – Task has been completed.

💬 Comments 
1. Add comments to tasks for notes and updates.

🔍 Filter and Sort 
1. Filter tasks based on Task Status.
2. Sort tasks based on Task Status for a better organization and easier navigation.
